# Getting started

This is a tutorial how to build a very first form with **React Form Component**.

## Installation

```
yarn add react-form-component
```


## Code steps

1. Wrap entitre app into `<FormThemeProvider>`. **Optionally add a theme** prop to define a [custom theme](http://localhost:6006/?path=/docs/docs-theming--page).

```jsx
// App.js
import React from 'react'
import ReactDOM from 'react-dom'
import { FormThemeProvider } from 'react-form-component'

const App = () =>
  <FormThemeProvider theme={{}}>
    Content of the app
  </FormThemeProvider>

ReactDOM.render(<App />, document.querySelector('#app'))
```

2. Put `Form` component anywhere in the app, declare field names in a `fields` prop - each unique for a form.
3. Use any input from the collection of default _Form UI Components_. Give each field a name prop, that corresponds with the one defined in `fields` prop of the ancestor `Form` component.
4. Use `FormButton` to trigger a submit function that returns all values formatted in a form of an object, that consists of keys declared as field names.

```jsx
// ExampleUserForm.js
import React from 'react'
import Form, {
  Input,
  Select,
  FormButton,
} from 'react-form-component'

const ExampleUserForm = () =>
  <Form fields={['name', 'email', 'type']}>
    <Input
      name='name'
      label='User name'
    />
    <Input
      name='email'
      type='email'
      label='E-mail'
    />
    <Select
      name='type'
      label='Type of a user'
      options={['Viewer', 'Moderator', 'Admin']}
    />
    <FormButton
      onClick={fields => console.log(fields)}
    >Save</FormButton>
  </Form>

export default BasicExampleForm
```
